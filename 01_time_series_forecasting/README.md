# Time-Series Forecasting System

## Business Problem
Forecast payment transaction volume and revenue so business teams can plan capacity, liquidity, pricing, and servicing needs.

## Methods
- Synthetic payment transaction data generation
- Lag features
- Rolling averages
- Calendar seasonality
- Random Forest forecasting model
- Forecast accuracy evaluation using MAE, RMSE, and MAPE

## Run
```bash
pip install -r requirements.txt
python forecasting_pipeline.py
```

## Resume Bullet
Built a time-series forecasting system to predict payment volume and revenue using lag features, rolling statistics, and machine learning models; evaluated forecast accuracy using MAE, RMSE, and MAPE to support revenue and capacity planning.
