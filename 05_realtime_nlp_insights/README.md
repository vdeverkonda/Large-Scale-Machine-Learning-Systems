# Real-Time NLP Insights Engine

## Business Problem
Analyze client servicing messages to identify what clients are asking about and help teams anticipate needs.

## Methods
- Simulated streaming messages
- TF-IDF feature extraction
- KMeans topic clustering
- Keyword extraction
- Simple sentiment/risk scoring

## Run
```bash
pip install -r requirements.txt
python nlp_insights.py
```

## Resume Bullet
Built a near real-time NLP insights engine that clusters client servicing messages, extracts top themes, and flags high-priority issues to help servicing teams anticipate client needs and improve support quality.
