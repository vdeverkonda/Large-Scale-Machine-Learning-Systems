# Dynamic Pricing Optimization Model

## Business Problem
Estimate the relationship between price and demand, then identify a revenue-maximizing price strategy.

## Methods
- Synthetic demand generation
- Price elasticity modeling
- Linear regression demand estimation
- Revenue optimization
- Scenario simulation under uncertainty

## Run
```bash
pip install -r requirements.txt
python pricing_optimization.py
```

## Resume Bullet
Developed a dynamic pricing optimization model that estimates price-demand relationships, simulates stochastic demand, and identifies revenue-maximizing pricing strategies under uncertainty.
