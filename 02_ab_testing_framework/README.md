# A/B Testing and Experimentation Framework

## Business Problem
Evaluate whether a new model, recommendation, or pricing strategy improves conversion or revenue before production rollout.

## Methods
- Simulated control and treatment groups
- Conversion-rate lift
- Revenue-per-user lift
- Two-proportion z-test
- Bootstrap confidence intervals
- Business impact estimation

## Run
```bash
pip install -r requirements.txt
python ab_test_framework.py
```

## Resume Bullet
Designed an A/B testing framework to evaluate model and product changes using statistical hypothesis testing, confidence intervals, and business impact estimation; simulated treatment effects and quantified conversion and revenue lift.
